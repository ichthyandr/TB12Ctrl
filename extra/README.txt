Каталог содержит данные для создания инсталлятора в NSIS.

1. Необходимо собрать программу в Release
2. Установить NSIS.
3. Создать каталог, скопировать в него exe файл программы, Logo.ico, 
	trash-o.ico, TBAccess.nsi (скрипт для NSIS)
4. Запустить шелл для используемой версии Qt (см. в меню Пуск для используемой системы сборки,
	например,  MinGW-7.3.0-64bits)
5. Перейти в каталог с exe файлом и выполнить деплой, например:
	C:\LIBS\Qt\Qt5.13.0\5.13.0\mingw73_32\bin\windeployqt.exe TBAccess.exe
6. Выполнить скрипт TBAccess.nsi с помощью NSIS. Будет создан файл инсталлятора.
